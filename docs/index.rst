Welcome to MGKMC's documentation!
=================================

MGKMC is an Athermal Quasi-Static (AQS) and Kinetic Monte Carlo (KMC) solver framework designed for simulating plasticity and deformation in amorphous materials via the Shear Transformation Zone (STZ) physical model. Fast, heavily vectorized via Numba, and flexible for various test scenarios such as Uniaxial Tension, Shear, and dynamic temperature behaviors.
